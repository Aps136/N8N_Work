# Smart Campus Automation System (n8n)

## Project Description
A multi-workflow automation system built using n8n to automate campus processes such as student registration, communication, reminders, feedback analysis, and reporting.

---

## Implemented Workflows
1. Student Registration & Validation  
2. Automated Communication  
3. Event Reminder Automation  
4. Feedback & AI Analysis  
5. Admin Reporting & Analytics  

All workflows are interconnected and use Webhooks, Cron jobs, Google Sheets, Email services, and AI APIs.

---

## Tech Stack
- n8n
- Google Sheets
- Gmail SMTP
- Gemini API
- Webhooks & Cron Jobs

---

## Key Features
- Event-driven automation  
- AI-based sentiment analysis  
- Automated email and reminder system  
- Weekly analytics reporting  
- Error handling and validation
